## 开发工具

- eclipse(或者 IntelliJ IDEA )

  - 多个项目可以在同一个窗口展示
  - 上课过程中不会使用到后台开发的框架

  - 版本 ≥ 1.8
  - [下载地址](https://www.oracle.com/technetwork/java/javase/downloads/index.html)

## 常用快捷键

### 代码提示 
- Mac : `Option + /` 
- Windows : `Alt + /`

### 错误修复 

- Mac : `Command + 1` 
- Windows : `Ctrl + 1`

### 快速生成代码

- Mac : `Option + Command + S`
- Windows : `Alt + Shift + S`

### 自动导入所需要的类 
- Mac : `Command + Shift + O`
- Windows : `Ctrl + Shift + O`
